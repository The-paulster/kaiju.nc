# Support

For bugs, feature requests, or publishing issues, please use the GitHub issue tracker:

https://github.com/The-paulster/kaiju.nc/issues

When reporting a problem, include:

- The KAIJU.NC version
- Your VS Code version
- The file extension and language mode shown in VS Code
- A small NC/G-code sample that reproduces the issue, if possible

## Safety

KAIJU.NC provides editor assistance only. It does not verify that CNC code is safe to run on a machine.

Always use proper simulation, setup checks, dry runs, and your shop's approved verification process before running any CNC program.
